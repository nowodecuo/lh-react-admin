# lh-react-admin

* Ant Design Pro React

### 目录

```

|-- src
    |-- assets 静态资源
    |-- components // 组件
        |-- LComponent // 自定义组件库
            |-- LIcon.tsx // 图标组件
            |-- LModalForm.tsx // 表单组件
            |-- LPasswordt.tsx // 密码修改组件
            |-- LTable.tsx // 表格组件
            |-- LUpload.tsx // 上传组件
    |-- config // 配置(枚举、数据结构等)
    |-- locales // 多语言配置
    |-- pages // 页面组件
    |-- services // 请求服务
    |-- typings // 接口及类型
    |-- utils // 工具方法

```
